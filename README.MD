# 🎮 GameVerse

> O universo definitivo de **Games & Animes** em um só lugar.

O **GameVerse** é uma aplicação web moderna que reúne os principais jogos e animes em um catálogo visual, com design futurista, animações suaves e experiência responsiva para qualquer dispositivo.

---

## 🚀 Demonstração

🔗 Em breve disponível via GitHub Pages

---

## ✨ Funcionalidades

* 🎮 Catálogo de jogos populares
* 📺 Catálogo de animes famosos
* ✨ Animações ao scroll (fade + interação)
* 🔔 Sistema de notificações (toast)
* 📱 Design totalmente responsivo
* 🎨 Interface moderna com tema gamer (neon + dark)
